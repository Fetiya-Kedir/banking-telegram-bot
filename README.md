# Banking Telegram Bot

A production-oriented multilingual Telegram banking assistant built with Python.

## Planned Features
- Multi-language support: English, Amharic, Afaan Oromo
- FAQ system
- Branch locator
- ATM locator
- Human support escalation
- About and contact sections
- PostgreSQL-backed persistence

## Tech Stack
- Python
- python-telegram-bot
- PostgreSQL
- SQLAlchemy
- Alembic
